## Overview

Voxi is an AI-powered mobile application designed to empower the **285 million visually impaired individuals** worldwide. By combining advanced artificial intelligence with intuitive voice interaction, Voxi transforms smartphones into intelligent assistive devices that provide real-time environmental understanding through audio feedback.

### Mission
To democratize accessibility technology and create independence through voice-controlled AI assistance, making everyday tasks accessible to visually impaired users at a fraction of the cost of traditional assistive devices.

## Voice Commands Reference

### Setup Commands
| English | Hindi | Function |
|---------|-------|----------|
| "English" | "English" | Switch to English language |
| "Hindi" | "हिंदी" | Switch to Hindi language |

| English | Hindi | Function |
|---------|-------|----------|
| "Read document" | "पढ़ो दस्तावेज़" | Read text from camera view |

## Installation & Usage

### System Requirements
- Android 7.0+ (API Level 24)
- Minimum 3GB RAM (4GB+ recommended)
- Camera and microphone permissions required
- 100MB available storage space
- Internet connection for initial setup only

### Installation Steps
1. Download APK from the link provided above
2. Enable "Install from Unknown Sources" in device settings
3. Install the application and grant required permissions
4. Launch app and wait for initial voice greeting
5. Select preferred language by saying "English" or "Hindi"
6. Begin using voice commands for text reading and object detection

### Usage Guidelines
- Ensure adequate lighting for optimal camera performance
- Position camera 1-3 feet from target objects or text
- Speak clearly and distinctly for accurate voice recognition
- Maintain steady camera position for 2-3 seconds during detection
- Use headphones in noisy environments for better audio feedback
